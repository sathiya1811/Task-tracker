# Task Tracker Application

## 📌 Project Overview

Task Tracker is a backend application developed using Spring Boot and MySQL to manage daily tasks. It allows users to create, update, delete, and view tasks efficiently.

---

## 🚀 Features

* Create new tasks
* View all tasks
* Update task details
* Delete tasks
* Filter tasks based on status

---

## 🛠️ Tech Stack

* Java
* Spring Boot
* Spring Data JPA
* MySQL
* REST APIs
* Postman

---

## ⚙️ API Endpoints

| Method | Endpoint    | Description   |
| ------ | ----------- | ------------- |
| GET    | /tasks      | Get all tasks |
| POST   | /tasks      | Add new task  |
| PUT    | /tasks/{id} | Update task   |
| DELETE | /tasks/{id} | Delete task   |

---

## 🗄️ Database

* MySQL database used
* Table: `tasks`

---

## ▶️ How to Run

1. Clone the repository
2. Open in Spring Tools Suite (STS)
3. Configure database in `application.properties`
4. Run the Spring Boot application
5. Test APIs using Postman
